### Adding a new article

1. Add an entry to `articles.json`:
```json
{
  "title": "...",
  "url": "https://twitter.com/...",
  "category": "HLD",
  "date": "YYYY-MM-DD",
  "description": "One line on what it covers."
}
```
2. Run `python3 generate_readme.py`
3. Commit both files.
